# Student Grade Management System

A simple, GUI-based desktop application designed to manage students, courses, and grades efficiently. This project allows users to record academic data, calculate averages, and export results through a user-friendly interface.

## 🚀 Features

- **Student Management:** Add new students and remove existing ones.
- **Course Management:** Add new courses and delete courses from the catalog.
- **Grade Processing:** Record grades for specific student-course pairs and edit them when necessary.
- **Calculations:** Automatically calculate the arithmetic average per student and convert it to a GPA format.
- **Data Display:** Show all grades, students, and courses in a clear, unified view.
- **Analytics & Ranking:** Rank students based on their average scores and show general course statistics (e.g., class average).
- **Export Data:** Export the displayed grades and statistics to a text or CSV file for external use.

## 🛠️ Technologies Used

- **Programming Language:** Python 3.x
- **User Interface (GUI):** Tkinter (Python built-in library)
- **Database:** SQLite (Python built-in library)

## 💻 How to Run the Project

Since this project uses Python's built-in libraries (Tkinter for GUI and SQLite for the database), there is no need to install external database servers or complex dependencies.

1. Ensure you have **Python 3.x** installed on your system.
2. Download or clone this project folder to your computer.
3. Open a terminal or command prompt in the project folder.
4. Run the main application file (assuming your main file is named `main.py`):

```bash
python main.py
